Summary: Latitude vs X

A Python script was used to visualize the weather of 500+ randomly selected cities across the world of varying distance from the equator. A series of scatter plots were then created to showcase the following relationships: Temperature (F) vs. Latitude, Humidity (%) vs. Latitude, Cloudiness (%) vs. Latitude, and Wind Speed (mph) vs. Latitude. This website provides the source data and visualizations that were created as part of the project.
